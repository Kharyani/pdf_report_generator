# PDF Report Generator (Python)

## 📌 Description

A Python-based CLI tool to generate professional PDF reports for student or company data using ReportLab.

## 🚀 Features

* Add data manually
* Load data from CSV/JSON
* Generate Student & Company reports
* Table formatting in PDF
* Automatic file saving
* Unique filenames
* Error handling

## 🛠 Requirements

* Python 3.x
* reportlab

Install dependencies:

```
pip install reportlab
```

## ▶️ How to Run

```
python main.py
```

## 📂 Project Structure

```
project/
│── main.py
│── data.json
│── students.csv
│── reports/
```

## 📄 Output

Generated PDFs are saved in the `reports/` folder.
